# Standard settings

I use the following standard settings and extensions for my projects.

## Installation

1. Copy the file `settings.json` into the directory `.vscode` in the root directory of your workspace.

1. Copy the file `extensions.json` into the directory `.vscode` in the root directory of your workspace.

    1. Go to the extensions.
    1. Enter the following command in the search bar: `@recommended`.
    1. Open the tab _Workspace recommendation_.
    1. Click on the cloud with the arrow on the right-hand side of the tab; the extensions will be installed.
    1. Some extensions require additional information, this must be added manually.
